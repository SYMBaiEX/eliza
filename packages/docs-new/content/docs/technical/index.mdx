## Core Concepts

### Agent Runtime
The heart of ElizaOS - a flexible runtime that manages agent lifecycle, memory, and interactions.

```typescript
import { AgentRuntime } from '@elizaos/core';

const runtime = new AgentRuntime({
  character: myCharacter,
  modelProvider: 'openai',
  plugins: [discordPlugin, twitterPlugin]
});
```

### Plugin System
Extend your agents with modular plugins that add new capabilities:

- **Actions**: Define what agents can do
- **Providers**: Supply contextual information
- **Evaluators**: Process and learn from interactions
- **Services**: Integrate with external platforms

### Memory Architecture
Advanced memory system with:
- Short-term conversational memory
- Long-term knowledge storage
- Relationship tracking
- Contextual embeddings

## Development Workflow

### 1. Install ElizaOS
```bash
bun install @elizaos/core @elizaos/cli
```

### 2. Create Your Agent
```typescript
import { Character } from '@elizaos/core';

export const myAgent: Character = {
  name: "TechBot",
  description: "A helpful technical assistant",
  modelProvider: "openai",
  plugins: [],
  settings: {
    voice: { model: "en-US-Neural2-F" },
    model: "gpt-4"
  }
};
```

### 3. Run Your Agent
```bash
bunx elizaos start --character ./myAgent.ts
```
